# Finley :open_book:
A terminal utility for encoding and decoding text files using [morse code.](https://en.wikipedia.org/wiki/Morse_code)


## Requirements

 - [rustup](https://rustup.rs) to install Rust :wink:
 - [Rust](https://www.rust-lang.org) **1.81.0** and above


## Recommended tools
 - [just](https://github.com/casey/just) **1.35** and above


## Getting Started

### Clone the repo and make it yours:

```bash
$ git clone git@github.com:diniremix/finley.git
```

### Build

```bash
$ cd finley
```

```bash
$ cargo build --release
# using just
$ just release
```
